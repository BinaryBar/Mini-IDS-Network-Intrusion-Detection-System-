# Mini Intrusion Detection System (IDS)

A lightweight Network Intrusion Detection System built using Python and Scapy.  
This system monitors network traffic and detects suspicious activities such as SYN scans and packet flooding attacks.

---

## Features

- Real-time packet sniffing
- SYN scan detection
- Flood attack detection
- Alert logging system
- Modular architecture

---

## Technologies Used

- Python
- Scapy
- TCP/IP Networking

---

## Project Structure

mini-intrusion-detection-system/

src/ → core IDS source code  
logs/ → alert log files  
screenshots/ → demo images  
docs/ → project documentation

---

## Installation

Install required packages:

pip install -r requirements.txt

---

## Run the Project

Run the IDS system:

python src/ids.py
